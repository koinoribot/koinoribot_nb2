# Koinoribot NB2 使用说明

Koinoribot NB2 是基于 NoneBot2 的综合娱乐插件包，主插件入口会自动加载 `plugins/` 目录下的子插件。当前 README 只说明本仓库中已启用、可使用的功能。

下文命令示例默认省略 NoneBot 命令前缀。若你的 Bot 配置了 `/`、`#` 等前缀，请按实际配置发送，例如 `/签到`、`#钓鱼`。

## 功能概览

- 签到、钱包、金币排行榜、UID 查询与跨平台账号绑定
- 炒股、转账、低保、红包、幸运游戏与幸运转盘
- 钓鱼、背包、卖鱼、放生、漂流瓶
- 宠物养成、扭蛋、宠物商店、技能、进化、誓约、排行榜
- 飞升之路、修炼、突破、修仙商店与升仙榜
- AI 画图、AI 修图、今日少女设定与人设图
- 猜单词、猜数字、猜日语、24 点小游戏
- 鲁迅说、莎士比亚说、自定义称呼、晚安
- SU 注册与公网白名单领养/审核

## 安装与加载

1. 将 `koinoribot_nb2` 放入 NoneBot2 项目的插件包目录中。
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 在 NoneBot2 项目中加载本插件包。按你的项目包名调整导入路径，例如：

```python
nonebot.load_plugin("plugins.koinoribot_nb2")
```

如果你的项目使用 `nonebot.load_plugins()` 或插件目录自动加载，也可以按项目现有方式加载。加载主插件后，`plugins/` 下的子插件会由 `koinoribot_nb2.__init__` 自动加载。

## 配置

主要配置集中在 `koinori_config.py`。

常用项：

- `qqbot_appid`、`qqbot_openid_api`：官方 QQBot 用户昵称/头像查询。
- `send_forward`：是否启用合并转发消息。
- `superusers`：最高权限 SU 的统一 UID 列表。
- `blackusers`：冻结用户 UID 列表。
- `dibao`、`gold_max`、`transfer_fee`、`min_rest`：经济系统参数。
- `cool_time`、`fish_cd`、`bait_price`、`bottle_price`、`comment_price`：钓鱼和漂流瓶参数。
- `gpt_image_api_key`、`gpt_image_api_base_url`、`gpt_image_model`：AI 画图/修图接口配置。
- `draw_cost`、`daily_limit`、`ai_draw_enable`：画图金币消耗、日限与开关。
- `public_bot`、`permit_bot`、`permit_group`、`ip_address`：公网白名单模式配置。

数据会自动写入 `src/database/koinoribot.db`。图片、字体等静态资源位于 `src/img` 和 `src/fonts`。

## 通用说明

- UID 是插件内部统一账号 ID，用于跨 OneBot V11 和 QQBot 保存金币、背包、宠物等数据。
- `注册验证码` 仅支持私聊使用。获取验证码后，到另一个平台发送 `绑定账号 <验证码>`，并按提示选择保留哪个 UID。
- SU 权限分为 level 0、level 1 和 level 2。`koinori_config.superusers` 中的 UID 视为 level 0；`注册su` 根据激活码类型注册为 level 1 或 level 2。
- 标注“SU”的命令需要 SU 权限。部分管理命令仅 level 0 可用。
- AI 画图和 AI 修图当前仅支持 OneBot 侧发送。
- 红包、24 点、领养云冰祈等功能仅支持群聊场景。

## 公网白名单与 SU

公网白名单用于云冰祈/多端接入场景。启动后会提供查询网页：

```text
http://<ip_address>:8888/
```

其中 `ip_address` 来自 `koinori_config.py`。

| 命令 | 用途 |
| :--- | :--- |
| `领养云冰祈` / `云冰祈` | 发起云冰祈领养申请 |
| `查询领养状态` / `我的领养` | 查询自己的申请和白名单状态 |
| `注销云冰祈` | 注销当前领养的云冰祈 |
| `审核列表` / `审核云冰祈` | SU 查看待审核申请 |
| `审核通过 <申请编号>` | SU 通过申请 |
| `审核拒绝 <申请编号> [理由]` | SU 拒绝申请 |
| `添加公网白名单 <主人QQ> <botQQ>` | SU 手动添加白名单 |
| `注册su <激活码>` | 使用激活码注册为 level 1 或 level 2 SU |

## 常见流程

### 跨平台绑定

1. 在账号 A 私聊发送 `注册验证码`。
2. 在账号 B 发送 `绑定账号 <验证码>`。
3. 按提示回复 `1` 或 `2`，选择保留哪个 UID。

### 发红包

1. 群内发送 `发红包 1000 5`。
2. 群成员发送 `抢红包` 领取。

### 宠物养成起步

1. 发送 `买宝石 10`。
2. 发送 `宠物商店` 查看可购买物品。
3. 发送 `购买 普通扭蛋 1`。
4. 发送 `开启 普通扭蛋`。
5. 抽到宠物后发送 `领养宠物 <名字>`。

### 修仙起步

1. 先在宠物系统完成 `永恒誓约`。
2. 发送 `宠物飞升` 推进飞升进度。
3. 飞升成功后通过 `修炼`、`突破`、`幸运突破` 提升境界。
4. 发送 `渡劫飞升` 尝试最终飞升。
